# Auralis Digital — Brand Guide

Date: 2026-04-24

## Official Brand Name
Auralis Digital

## Primary Service Positioning
Professional websites for local businesses.

## Core Value Proposition
Helping businesses build trust, attract customers, and grow online.

## Tagline
Built for Trust, Growth, and Action.

## Brand Architecture
- Parent Brand: Auralis Digital
- Current Service: Website Design & Growth Support
- Future AI Layer: Nova by Auralis
- Future Products: Auralis Tools / Auralis Systems

## Voice & Tone
- Professional
- Clear
- Practical
- Modern
- Trustworthy
- No hype

## Messaging Rules
Sell outcomes, not features.
Focus on:
- More calls
- More quote requests
- Better first impressions
- Easier customer action
- Stronger online trust

## Immediate Rebrand Targets
- Homepage brand name
- README messaging
- Outreach scripts
- Proposal template
- Footer/contact copy
