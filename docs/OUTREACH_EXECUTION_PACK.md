# Outreach Execution Pack

## Daily Minimum Standard
- 10 new prospects researched
- 5 personalized messages sent
- 5 follow-ups sent
- Track all activity

## Message Structure
1. Quick relevance note
2. Problem noticed
3. Simple value offer
4. Easy next step

## Follow-Up Cadence
- Day 1 initial message
- Day 3 follow-up
- Day 7 final follow-up

## Scoreboard
Track replies, calls, quotes, closes.
