# Trust & Credibility Audit Response — 2026-04-25

I read the pasted review. It is harsh, but it is useful because it shows exactly how a cautious outsider might judge Auralis Digital right now. The main flagged issues were: brand-new domain, Gmail contact email, no public reviews, no verifiable business presence, unclear portfolio proof, and possible HTTPS/security-header weakness.

My take: **this does not mean the business is bad or a scam.** It means the site currently looks like a very new solo business, and new solo businesses need extra trust signals fast.

## What to fix first

Highest priority corrections:

1. **Use a domain email**
   - Create something like `hello@auralisdigital.net` or `contact@auralisdigital.net`.
   - Keep Gmail as backend forwarding if needed, but do not display Gmail as the main public email.

2. **Add a trust / legitimacy section**
   - Founder name
   - Michigan-based
   - Direct phone number
   - Clear service area
   - "Founder-led, small-business web design"
   - Simple statement that Auralis Digital is newly launched

3. **Add real portfolio proof**
   - Keep the example projects, but label them clearly:
     - "Demo concept"
     - "Sample build"
     - "Example website preview"
   - Add what each example is meant to prove.

4. **Add contract / process clarity**
   - Scope
   - Deposit/payment structure
   - Revision process
   - Launch process
   - Refund/change policy basics

5. **Fix HTTPS / security confidence**
   - Confirm GitHub Pages is enforcing HTTPS.
   - Add any available security improvements.
   - At minimum, make sure all links use `https://www.auralisdigital.net`.

6. **Add testimonials later**
   - Do not fake them.
   - For now, add "New business — references available after first completed projects" or keep proof as "example work."

## Best response strategy

Do **not** hide that the business is new. Use it honestly:

> Auralis Digital is a newly launched, founder-led web design service based in Michigan. The focus is simple: clear, affordable websites for local businesses that need a stronger first impression and easier customer contact.

That turns the "brand-new" issue into transparency instead of a red flag.
